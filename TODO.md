# QuickSnippet 开发计划

## 🚀 核心功能增强

- [ ] **标签/分类系统**
  - 给代码片段打标签（如 `docker`、`git`、`网络`）
  - 搜索框支持 `#标签` 语法过滤
  - 比语言类型更灵活，适合日常管理

- [ ] **编辑功能增强**
  - 语法高亮编辑器（Monaco Editor 或 CodeMirror）
  - 代码格式化（Prettier 等）

- [x] **数据导入/导出**
  - 导出为 JSON / Markdown 文件
  - 从文件批量导入
  - 方便备份和迁移

## 🎨 UI 体验

- [ ] **QuickLaunch 增强**
  - 搜索结果中显示匹配的关键词高亮
  - 预览代码行数更多的片段
  - 最近使用的排前面
  - 支持中文拼音模糊搜索

- [ ] **代码片段预览优化**
  - 列表项显示更多信息（创建时间、字符数）

## 🔧 系统集成

- [x] **系统托盘**
  - 最小化到系统托盘图标
  - 右键菜单：打开主界面 / QuickLaunch / 退出
  - 关闭窗口自动最小化到托盘

- [x] **文件夹分组**
  - 侧边栏文件夹列表
  - 创建、重命名、删除文件夹
  - 按文件夹筛选代码片段
  - 编辑器选择文件夹

- [ ] **剪贴板历史**（进阶）
  - 监听系统剪贴板
  - 自动保存复制的文本到代码片段库

## ⚡ 数据层

- [ ] **云同步**
  - 架构已预留同步字段
  - 可对接 WebDAV / iCloud / 自建服务
  - 多设备共享代码片段

- [ ] **统计与搜索优化**
  - 复制次数统计
  - 常用片段推荐
  - 搜索历史
